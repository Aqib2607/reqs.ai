## 🎯 About Reqs.ai

Transform your project ideas into structured, actionable documents in seconds. Reqs.ai leverages advanced AI to help developers, product managers, and teams create comprehensive project plans and Product Requirement Documents (PRDs) effortlessly.

- 🔭 **What it does:** Generates detailed project plans and PRDs from natural language descriptions
- 🌱 **Built for:** Developers, Product Managers, and Startup Teams
- 💼 **Features:** AI-powered generation, customizable templates, version tracking, and export options
- 💬 **Tech Stack:** React, TypeScript, Laravel, and Gemini AI
- 📄 **Live Demo:** [Coming Soon](https://reqs.ai)
- ⚡ **Fun fact:** This README was generated with the same attention to detail we put into every PRD!

## 🚀 Getting Started

### Prerequisites

- Node.js (v18 or higher)
- PHP (v8.1 or higher)
- Composer
- npm or yarn

### Installation

1. **Clone the repository:**
   ```sh
   git clone https://github.com/Aqib2607/reqs.ai
   cd reqs.ai
   ```

2. **Install frontend dependencies:**
   ```sh
   npm install
   ```

3. **Install backend dependencies:**
   ```sh
   cd backend
   composer install
   ```

4. **Set up environment variables:**
   ```sh
   cp .env.example .env
   # Configure your database and API keys
   ```

5. **Run migrations:**
   ```sh
   php artisan migrate
   ```

6. **Start the development servers:**
   ```sh
   # Frontend (from root)
   npm run dev
   
   # Backend (from backend directory)
   php artisan serve
   ```
